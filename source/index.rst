=====================================================
欢迎来到CSU软件工程专业保研攻略
=====================================================

----
前言
----
由中南软工学长学姐编写的保研攻略，主要覆盖了中南软工的同学常去的目标学校和院系。

希望大家都能更快更高效更少踩坑地成功保研，拥有光明的未来。

----
说明
----
本文档分为三个部分：

- 第一部分为一些通用的建议，如录取情况，保研流程，文书准备，其他建议等；
- 第二部分为为各个院校说明，主要介绍 **学长学姐参与过的** 每个院校和院系的夏令营或者预推免经验；；
- 第三部分为作者信息和联系方式。
